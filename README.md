# To-Do List Application

A simple and interactive To-Do List application built with HTML, CSS, and JavaScript. This application allows users to add, remove, and mark tasks as complete.

## Features

- Add new tasks
- Remove tasks
- Mark tasks as complete
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
